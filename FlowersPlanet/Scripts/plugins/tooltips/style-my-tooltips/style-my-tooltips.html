Style-my-tooltips demo

Style native browser tooltips easily with Style-My-Tooltips jQuery plugin. A very simple script to enhance the look of tooltips. It works just like browser's native tooltips with few options and styling via CSS. By default, the script affects any element with a "title" attribute like links, paragraphs, images etc. but you can easily set it to affect only specific classes.
